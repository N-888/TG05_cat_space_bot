# TG05 CatSpace Bot

Современный Telegram-бот на Python 3.14 и aiogram 3.x, который объединяет сразу два внешних API в одном красивом проекте:

- TheCatAPI — поиск информации и изображений кошек по породе;
- NASA APOD API — получение космической картинки дня по текущей, случайной или выбранной дате.

## Этот проект создан для:

- учебной работы;
- демонстрации работы с внешними API;
- показа навыков структуры проекта, FSM и обработки ошибок.

---

## Почему проект удачный

TG05 CatSpace Bot сделан как аккуратный, понятный и расширяемый проект с оптимальной структурой папок и файлов.

- современная архитектура на aiogram 3.x;
- раздельные обработчики команд;
- отдельные сервисы для внешних API;
- хранение секретов в `.env`, а не в коде;
- inline-кнопки и красивое меню;
- пошаговый ввод породы через FSM;
- обработка ошибок без падения приложения;
- понятный и чистый интерфейс сообщений;
- готовая структура для дальнейшего развития.

## Сильные стороны реализации

- не хранит ключи в коде;
- легко расширяется новыми командами;
- читабельный код по модулям;
- подходит для запуска локально на Windows через PyCharm;
- визуально приятный и понятный интерфейс бота;
- хорошая основа для будущего деплоя.

##  Возможности TG05 CatSpace Bot 

### Работа с кошками

- `/cat_random` — случайный котик;
- `/cat_breed` — пошаговый поиск породы;
- `/cat_breed Bengal` — быстрый поиск породы одной командой;
- вывод:
  - фото;
  - название породы;
  - генетическое происхождение / родословная;
  - темперамент / характер;
  - продолжительность жизни (примерно);
  - описание;
  - ссылка на Wikipedia, если доступна.

### Работа с космосом

- `/space_today` — APOD за сегодня;
- `/space_random` — случайная APOD за последний год;
- `/space_date YYYY-MM-DD` — APOD по выбранной дате;
- корректная обработка как изображений, так и других типов медиа.

### Интерфейс

- красивое стартовое меню;
- inline-кнопки;
- понятные эмодзи;
- лаконичные и дружелюбные ответы;
- удобная навигация назад в меню.

### Команды бота

- /start
- /menu
- /help
- /cat_random
- /cat_breed
- /cat_breed Bengal
- /space_today
- /space_random
- /space_date 2025-12-31
